# Sistema de Tipografía - AutoChain

## Jerarquía de Títulos

### H1 - Títulos Principales de Pantalla
- **Tamaño**: `text-[28px]`
- **Peso**: `font-bold`
- **Line Height**: `leading-[34px]`
- **Color**: `text-slate-800` o `text-transparent bg-clip-text bg-gradient-to-r from-indigo-600 to-purple-600` (para pantallas especiales como Login)
- **Uso**: Título principal de cada pantalla
- **Ejemplo**: "Mis Vehículos", "Detalles de Mantenimiento"

### H2 - Títulos de Sección
- **Tamaño**: `text-[20px]`
- **Peso**: `font-bold` o `font-semibold`
- **Color**: `text-slate-700`
- **Uso**: Secciones dentro de una pantalla
- **Ejemplo**: "Items de Trabajo", "Fotografías", "Documentos"

### H3 - Subtítulos y Nombres de Cards
- **Tamaño**: `text-[16px]` - `text-[18px]`
- **Peso**: `font-semibold` o `font-bold`
- **Color**: `text-slate-700` o `text-slate-900`
- **Uso**: Títulos en cards, nombres de vehículos, subtítulos

### Body Text
- **Tamaño**: `text-[14px]` - `text-[15px]`
- **Peso**: `font-normal` o `font-medium`
- **Color**: `text-slate-600` (normal), `text-slate-500` (secundario), `text-slate-400` (placeholder)

### Labels
- **Tamaño**: `text-[14px]` - `text-[15px]`
- **Peso**: `font-medium`
- **Color**: `text-slate-700`

## Pantallas Especiales

### Login & Register
- **H1**: `text-[40px] font-black` con gradiente
- **Line Height**: `leading-[48px]`
- Uso de gradiente para destacar la marca

## Notas
- Evitar usar tamaños arbitrarios fuera de este sistema
- Los tamaños están optimizados para pantallas móviles de 414px de ancho
- Mantener consistencia en toda la aplicación
